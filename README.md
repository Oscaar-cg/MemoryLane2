# MemoryLane 📱

MemoryLane is an Android application that allows users to create and manage personal memory capsules.

##  Features
- Create memory capsules with title, description, and date
- Add photos from gallery
- View images in a carousel (ViewPager2)
- Clean and intuitive user interface
- Navigation using activities and fragments
